# Tutorials

A collection of tutorials demonstrating various workflows and automation techniques.

## Available Tutorials

### [n8n Support Ticket Evaluator](./n8n-support-ticket-evaluator/)
Learn how to build an automated evaluation workflow using n8n to classify and evaluate customer support tickets using LLMs. This tutorial covers:
- Setting up n8n workflows
- Integrating with Hugging Face datasets
- Using Ollama API for LLM inference
- Google Sheets integration for evaluation data
- Calculating classification accuracy metrics

## Structure

Each tutorial is organized in its own directory with:
- `README.md` - Complete tutorial documentation
- Workflow files (JSON, YAML, etc.)
- Configuration examples
- Sample data files
- Any other tutorial-specific resources

## Contributing

When adding a new tutorial:
1. Create a new directory with a descriptive name (kebab-case)
2. Include a comprehensive `README.md` in the tutorial directory
3. Add an entry to this main README
4. Follow the existing structure and documentation style
